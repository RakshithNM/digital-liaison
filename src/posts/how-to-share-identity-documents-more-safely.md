When documentation support involves PAN, passport, or driving licence records, the safest workflow is the one that asks for the least amount of information needed to move the matter forward.

## Simple habits that help

- Share only the documents that are reasonably necessary for the specific task.
- Prefer partial document details where the full identifier is not required for the current step.
- Avoid sending the same sensitive file in multiple channels if one secure route is enough.
- Keep a note of who received the documents and why.

## Before you send anything

Ask these questions first:

1. Is this document actually required for the current step?
2. Can a masked or partial version work instead?
3. Is the recipient clearly identified?
4. Is there an official portal where the document should be uploaded directly instead?

## Use the official source when possible

If a task can be completed directly through an official authority, use that authority first. For PAN-related services, refer to [Protean PAN Services](https://tinpan.proteantech.in/services/pan/pan-index.html). For voter-related services, use the [Voters' Service Portal](https://voters.eci.gov.in/). For passport services, use [Passport Seva](https://portal2.passportindia.gov.in/AppOnlineProject/welcomeLink).

Private assistance can still be useful for review and preparation, but it should not widen the amount of personal data being circulated without a clear reason.
